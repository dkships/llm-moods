# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

If you experience or witness unacceptable behavior, please open a GitHub Issue or contact the maintainer directly.

## Enforcement

Project maintainers will review and investigate all complaints and will respond in a way that they deem appropriate to the circumstances.
